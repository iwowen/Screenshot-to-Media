Screenshot To Media 能够将截图到内容粘贴到媒体库。

== Description ==

在编辑文章时，都希望能够直接粘贴截图到媒体库，而不是先保存再上传图片了。
wordpress自带到编辑器已经支持将截图复制到媒体库，但是如果换成其他到编辑器就不行。所以就需要用到这个插件。
Screenshot To Media 能够将截图到内容粘贴（ctrl + v）到媒体库，这样就能很方便的编辑文章了。

使用方法：
首先可以截图，或者在浏览器图片上右键复制图片，或者mac上复制本地图片。然后在媒体库页面按住（ctrl + v），图片将上传到媒体库。

支持此功能的页面：
- 媒体添加页面
- 媒体库方块图片展示页面
- 文章编辑需打开媒体库弹窗
- 文章新增需打开媒体库弹窗

支持网站：https://www.iwowen.cn/ [什码到博客](https://www.iwowen.cn/ "什码到博客")
